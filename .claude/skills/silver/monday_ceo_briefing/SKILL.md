# Monday CEO Briefing Generator Skill

## Overview

**Skill Name:** `monday_ceo_briefing`
**Domain:** `silver`
**Purpose:** Generate a comprehensive weekly CEO briefing covering revenue, bottlenecks, subscriptions, suggestions, and pending approvals. Reads vault context over a 7-day window and saves to `/Briefings/YYYY-MM-DD_Monday.md`, updating Dashboard.

**Core Capabilities:**
- 7-day data aggregation (Company_Handbook, /Done, /Accounting, /Pending_Approval)
- Executive summary with week-over-week activity
- Revenue summary from accounting + completed invoices
- Bottleneck & blocker identification (idle days, goal gaps)
- Subscription audit (recurring payments detection)
- AI-generated suggestions & next steps
- Pending approvals overview with count and details
- Dashboard auto-update (Weekly Briefings section)
- Comprehensive logging

## Data Sources

### 1. Business Goals
**Source:** `Company_Handbook.md`

Extracts:
- Mission statements
- Strategic objectives
- Active goals from "## Goals" or "## Objectives" sections

### 2. Completed Tasks
**Source:** `/Done/*.md` (last 7 days)

Extracts:
- Task title and metadata
- Completion timestamp
- Task type (email, file_drop, whatsapp, etc.)
- Priority level
- Content preview (first 500 chars)

### 3. Accounting Data
**Source:** `/Accounting/*.md`

Extracts:
- Transaction records with amounts
- Revenue signals ($, revenue, invoice, paid keywords)
- Subscription/recurring payment detection

### 4. Pending Approvals
**Source:** `/Pending_Approval/*.md`

Extracts:
- Item title, type, priority
- Count of pending items

## Output Structure

### Briefing File: `/Briefings/YYYY-MM-DD_Monday.md`

```markdown
---
briefing_id: "20260301_weekly"
generated_at: "2026-03-01T23:00:00Z"
type: monday_ceo_briefing
period: 7d
lookback_days: 7
---

# Monday CEO Briefing — 2026-03-01

## 1. Executive Summary
## 2. Revenue Summary
## 3. Bottlenecks & Blockers
## 4. Subscription Audit
## 5. Suggestions & Next Steps
## 6. Pending Approvals Overview

---
*Generated by AI Employee Monday CEO Briefing Generator*
```

### Dashboard Update

Adds "Weekly Briefings" section with links:

```markdown
## Weekly Briefings

- [[2026-03-01_Monday.md]]
```

## Configuration

| Env Var | Default | Description |
|---------|---------|-------------|
| `VAULT_PATH` | `./AI-Employee-Vault` | Path to vault directory |
| `DRY_RUN` | `false` | Preview mode (no writes) |
| `LOG_LEVEL` | `INFO` | Logging verbosity |
| `BRIEFING_LOOKBACK_DAYS` | `7` | Days of data to include |

## Usage

```bash
# Generate Monday CEO briefing
python monday_ceo_briefing.py

# Preview mode
DRY_RUN=true python monday_ceo_briefing.py

# Debug mode
LOG_LEVEL=DEBUG python monday_ceo_briefing.py
```

## Scheduling

Triggered Sunday at 11:00 PM so the briefing is ready for Monday morning.

```cron
0 23 * * 0 /path/to/AI-Employee/scripts/monday_ceo_briefing.sh
```
