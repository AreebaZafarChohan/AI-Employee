# Daily Briefing Generator Skill

## Overview

**Skill Name:** `daily_briefing_generator`
**Domain:** `silver`
**Purpose:** Read vault context (Business_Goals, completed tasks, accounting data), generate comprehensive daily briefing with executive summary, revenue snapshot, task summary, bottlenecks, and suggestions. Save to `/Briefings/YYYY-MM-DD_Daily.md` and update Dashboard.

**Core Capabilities:**
- Multi-source data aggregation (Company_Handbook, /Done, Accounting)
- Executive summary generation
- Revenue snapshot compilation
- Task summary with metadata
- Bottleneck identification
- Actionable suggestions
- Dashboard auto-update
- Comprehensive logging

## Data Sources

### 1. Business Goals
**Source:** `Company_Handbook.md`

Extracts:
- Mission statements
- Strategic objectives
- Active goals from "## Goals" or "## Objectives" sections

### 2. Completed Tasks
**Source:** `/Done/*.md` (last 24 hours)

Extracts:
- Task title and metadata
- Completion timestamp
- Task type (email, file_drop, whatsapp, etc.)
- Priority level
- Content preview (first 500 chars)

### 3. Accounting Data
**Source:** `/Accounting/*.md` and vault root

Extracts:
- Transaction records
- Payment requests
- Revenue signals ($, revenue, invoice, paid keywords)
- Pending payments

## Generation Logic

### Executive Summary

```python
def generate_executive_summary(business_goals, tasks):
    - List first 5 business goals
    - Count total tasks completed
    - Break down by task type
    - Include timestamp
```

### Revenue Snapshot

```python
def generate_revenue_snapshot(accounting):
    - List up to 5 recent transactions
    - Show amounts if available
    - Extract revenue signals from document snippets
    - Handle missing data gracefully
```

### Task Summary

```python
def generate_task_summary(tasks):
    - Limit to 10 most recent tasks
    - Format title (replace - with spaces, title case)
    - Show completion date
    - Preview first 5 lines of content
```

### Bottlenecks

```python
def generate_bottlenecks(tasks, business_goals):
    - Count high priority items
    - Count pending approval items
    - Compare task count vs goal count
    - Flag misalignments
```

### Suggestions

```python
def generate_suggestions(tasks, business_goals):
    - Always suggest reviewing pending items
    - Suggest goal alignment
    - Suggest email follow-ups if emails processed
    - Suggest weekly review if >= 5 tasks completed
```

## Configuration

| Env Var | Default | Description |
|---------|---------|-------------|
| `VAULT_PATH` | `./AI-Employee-Vault` | Path to vault directory |
| `DRY_RUN` | `false` | Preview mode (no writes) |
| `LOG_LEVEL` | `INFO` | Logging verbosity |
| `BRIEFING_LOOKBACK_HOURS` | `24` | Hours of tasks to include |

## Usage

```bash
# Generate daily briefing
python daily_briefing_generator.py

# Preview mode (no file writes)
DRY_RUN=true python daily_briefing_generator.py

# Custom lookback period
BRIEFING_LOOKBACK_HOURS=48 python daily_briefing_generator.py

# With logging
LOG_LEVEL=DEBUG python daily_briefing_generator.py
```

## Error Handling

### Missing Data Sources
- Company_Handbook not found → Empty goals list
- No tasks in /Done → "No tasks completed" message
- No accounting files → "No recent transactions" message

### File Write Failures
- Log error with details
- Return non-zero exit code
- Do not update Dashboard if briefing save fails

### Dashboard Update Failures
- Log warning
- Continue (briefing already saved)
