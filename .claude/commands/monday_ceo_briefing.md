# Monday CEO Briefing Generator

You are executing the **Monday CEO Briefing Generator** skill. Follow every step precisely.

## Step 1 — Read Data Sources

### 1.1 Business Goals
Read `AI-Employee-Vault/Company_Handbook.md`:
- Extract sections with "Goal", "Objective", or "Mission" in headers
- List all bullet points under these sections
- Maximum 5 goals for summary

### 1.2 Completed Tasks (7-day window)
Read `AI-Employee-Vault/Done/*.md` files from last 7 days:
- Check `completed_at`, `executed_at`, `created_at` fields in frontmatter
- Fall back to file modification time
- Extract: title, type, priority, body preview (first 500 chars)

### 1.3 Accounting Data
Read `AI-Employee-Vault/Accounting/*.md` (if exists):
- Extract transaction metadata (amount, recipient, date)
- Find revenue signals ($, revenue, paid, invoice keywords)
- Detect subscriptions (subscription, recurring, monthly, annual keywords)

Also scan `AI-Employee-Vault/Done/*.md` for invoice/payment files within the 7-day window.

### 1.4 Pending Approvals
Read `AI-Employee-Vault/Pending_Approval/*.md`:
- Extract title, type, priority from each file
- Count total pending items
